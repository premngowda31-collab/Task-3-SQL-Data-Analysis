# Task 3: SQL for Data Analysis

## Objective

To extract and analyze data from an Ecommerce database using SQL queries.

## Tools Used

* MySQL Workbench

## Database Description

The database consists of the following tables:

* Customers
* Orders
* Products
* Order Details

## SQL Operations Performed

* SELECT, WHERE, ORDER BY
* GROUP BY with aggregate functions (SUM, AVG)
* INNER JOIN and LEFT JOIN
* Subqueries
* View creation
* Index optimization

## Interview Questions

1. **Difference between WHERE and HAVING**

   * WHERE filters rows before aggregation
   * HAVING filters data after GROUP BY

2. **Types of Joins**

   * INNER JOIN
   * LEFT JOIN
   * RIGHT JOIN
   * FULL JOIN

3. **Average Revenue Per User**

   * Calculated as total revenue divided by number of users

4. **Subqueries**

   * A query nested inside another query

5. **Query Optimization**

   * Use indexes
   * Avoid SELECT *
   * Optimize joins

6. **View in SQL**

   * A virtual table created from a query

7. **Handling NULL Values**

   * Use IFNULL() or COALESCE()
